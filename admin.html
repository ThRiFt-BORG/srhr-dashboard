Attach Policy Documents

Link a document URL to a policy so a download button appears on the county page. Host your PDFs on Google Drive, Dropbox, or in the docs/ folder.

How to host documents free

Options for making PDFs accessible:
1
Google Drive: Upload PDF → right-click → Share → Anyone with link → Copy link. Paste that URL above.
2
GitHub: Place PDF in the docs/ folder → push to GitHub → use the raw GitHub URL: https://raw.githubusercontent.com/ThRiFt-BORG/srhr-dashboard/main/docs/filename.pdf
3
Dropbox: Share link → change ?dl=0 to ?dl=1 at end of URL for direct download.

📊 Google Sheets Integration

Connect a Google Sheet as a live data source so non-technical staff update data in a familiar spreadsheet interface.
How it works:
You publish a Google Sheet as a CSV. The dashboard reads it every time a page loads. Staff update the sheet → portal updates automatically. No coding, no file editing.

Setup Steps

1
Make a copy of the EACHRights SRHR Data Template sheet:
📋 Open Template Sheet →
(Template link — paste your Sheet URL below after copying)
2
In your copy: File → Share → Publish to web
Choose Sheet: Policies and format: CSV → Click Publish → Copy the URL
3
Paste the published CSV URL for each tab below and click Connect.
